# AI 高频禁用词表 v1.0

> 以下词汇在 AI 生成文本中出现频率远高于人类作者，容易被读者/审稿人识别为 AI 痕迹。润色时必须替换。

| ----------------------- | ---------------------------------- |
| leverage / leveraged    | use / apply / applied              |
| delve / delve into      | examine / investigate / study      |
| crucial                 | important / key / essential        |

| facilitate              | enable / allow / support           |
| elucidate               | clarify / explain / reveal / show  |
| underscore              | highlight / emphasize / show       |

| novel                   | new（除非确实首次报道）                      |
| cutting-edge            | advanced / recent                  |
| state-of-the-art        | current / advanced / latest        |
| groundbreaking          | 删除，或 new / first                   |
| landscape               | field / area / context             |
| paradigm                | approach / framework / model       |
| realm                   | field / area / domain              |
| interplay               | interaction / relationship         |
| notably                 | 删除，或 in particular                 |
| furthermore             | 删除，或句间逻辑衔接                         |
| moreover                | 删除，或句间逻辑衔接                         |
