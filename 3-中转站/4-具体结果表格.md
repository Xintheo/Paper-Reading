## 文献1 (PMID: 40049228) 研究结果一览

| 研究目的 | 涉及方法 | 关键发现 |
|---|---|---|
| 鉴定介导神经酰胺激活炎症小体的膜受体 | G蛋白解离实验、ELISA、钙流检测、流式凋亡、细胞转染、单细胞转录组学基础分析 | 从122个Gq偶联GPCR中鉴定出CYSLTR2和P2RY6作为C16:0神经酰胺的功能性受体；二者通过Gq信号介导神经酰胺诱导的炎症小体激活和IL-1β产生，双敲除或联合拮抗可阻断该效应 |
| 鉴定CYSLTR2和P2RY6对神经酰胺碳链长度的选择性谱 | G蛋白解离实验 | CYSLTR2仅被中长链神经酰胺(C16:0/C18:0/C20:0)激活，而P2RY6还可被超长链神经酰胺(C22:0/C24:0)激活，两受体呈现差异化的配体识别谱 |
| 监测神经酰胺结合诱导的受体胞外构象变化 | FlAsH-BRET检测 | C16:0神经酰胺结合诱导CYSLTR2和P2RY6的N端分别远离ECL2和ECL1，揭示两受体具有共同但不同的构象激活机制 |
| 验证CYSLTR2和P2RY6的体内表达及配体直接结合 | 免疫荧光、流式凋亡、FlAsH-BRET检测 | 构建的FlAsH-BRET敲入小鼠证实两受体在主动脉内皮细胞和巨噬细胞中表达，C16:0神经酰胺可在体内直接结合并诱导受体构象变化 |
| 验证CYSLTR2和P2RY6介导神经酰胺加重动脉粥样硬化 | 动物模型、油红O染色、免疫组化、血清生化检测 | CYSLTR2和P2RY6协同介导神经酰胺加重的动脉粥样硬化：单基因敲除部分减轻斑块面积，双基因敲除显著消除神经酰胺的致动脉粥样硬化效应 |
| 验证神经酰胺-CYSLTR2-P2RY6轴在CKD加重动脉粥样硬化中的作用 | UPLC-MS分析、统计分析、动物模型、油红O染色、血清生化检测 | CKD患者血浆长链神经酰胺升高且与Gensini评分正相关；5/6肾切除小鼠模型中，Cysltr2/P2ry6基因敲除或拮抗剂均可减轻CKD加重的动脉粥样硬化，不改变血浆脂质水平 |
| 解析神经酰胺-CYSLTR2-Gq复合物的冷冻电镜结构 | 冷冻电镜数据采集与处理、模型搭建与精修 | 解析C16:0和C20:0神经酰胺-CYSLTR2-Gq复合物分别达2.7 Å和2.8 Å分辨率，揭示神经酰胺通过斜向通道样配体口袋结合，区别于其他A类GPCR的激活机制 |
| 解析CYSLTR2鞘氨醇链结合通道的结构功能关系 | 模型搭建与精修、G蛋白解离实验、分子动力学 | SCBC通道内连续酪氨酸(Y116/Y119/Y123)通过π-π堆积识别双键，N202和R267形成极性网络锚定鞘氨醇骨架；关键残基突变显著削弱受体激活 |
| 解析CYSLTR2脂肪酸链结合口袋的结构可塑性 | G蛋白解离实验、模型搭建与精修、分子动力学、分子对接 | FABP口袋通过结构可塑性区分不同碳链长度：C16:0结合HK口袋(H270/K275)，C20:0结合FYL口袋(F41/Y98/L291)；这种疏水可塑性是CYSLTR2识别特定神经酰胺亚型的决定因素 |
| 解析神经酰胺与P2RY6的潜在结合模式 | 分子动力学、分子对接、G蛋白解离实验 | 分子动力学模拟结合突变验证表明，C16:0神经酰胺以不同于CYSLTR2的结合模式与P2RY6相互作用，验证了其结构特异性的配体识别机制 |

## 文献2 (PMID: 42099453) 研究结果一览

| 研究目的 | 涉及方法 | 关键发现 |
|---|---|---|
| 绘制主动脉夹层病变的单细胞转录组图谱并揭示血栓相关的细胞和分子特征 | 单细胞转录组学基础分析、GSEA富集分析、富集分析-GO和KEGG | scRNA-seq鉴定出5个SMC亚群，血栓阳性样本表现为SMC3和SMC4比例升高而SMC2几乎缺失，M2巨噬细胞富集；血栓阳性SMCs富集缺氧应激通路，血栓阴性则富集细胞骨架相关通路 |
| 揭示主动脉夹层中SMC亚群的功能表型及其微环境调控机制 | 单细胞转录组学基础分析、GSEA富集分析、qRT-PCR、Transwell检测、Western Blot、免疫荧光、细胞培养、细胞转染 | SMC2呈促纤维化表型(上调胶原及ECM重塑因子)，SMC3呈MAPK依赖性促炎症表型(分泌趋化因子招募单核细胞)；细胞通讯分析显示VEGFA信号从SMC向成纤维细胞转移，体外共培养验证SMC2诱导内皮间质转化而SMC3促进单核细胞迁移 |
