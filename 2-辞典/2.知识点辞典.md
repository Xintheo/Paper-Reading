# 知识点辞典

以限定词为单位归纳具象化知识，跨章节共用。适用章节：引言、摘要、结果、讨论、综述。

## 命名规则

知识点名称 = 限定词 + 相关知识

限定词使用中文全称，不使用英文缩写。同一限定词只建一个知识点笔记，内部按元素名分二级标题。

- 研究对象类 → 疾病中文全称、因素中文名称、分子中文名称
- 研究方法类 → 方法中文名称，取核心方法名，不堆叠修饰词
- 治疗手段类 → 药物或干预措施中文名称

示例：
- 酒精性肝病相关知识
- 特发性肺纤维化相关知识
- 孟德尔随机化相关知识
- 网络毒理学相关知识
- 白藜芦醇相关知识

## 维度规则

每个知识点归属一个或多个维度，由段落叙事角度决定：

| 维度 | 定义 |
|------|------|
| 研究对象 | 段落核心聚焦某个疾病、因素、分子、药物的属性、机制或关联 |
| 研究方法 | 段落核心聚焦某项技术、方法、分析策略的工具能力和应用 |
| 治疗手段 | 段落核心聚焦某个药物或干预措施的临床价值、疗效或开发前景 |

同一段落可同时属于多个维度。若以上三维度无法覆盖，可新建维度并在新发现中登记。

## 逻辑链属性

逻辑链取值为引言写作辞典中的标准名称。同一知识点在多个逻辑链中出现时，列表累积去重。

## 归一化规则

- 限定词相同 → 归入同一知识点，不新建
- 限定词语义相近 → 归一化到已有名称
- 限定词必须使用中文全称，不使用英文缩写
- 方法类限定词只取核心方法名，不堆叠修饰词

## 生成规则

段落中至少出现一个具象知识元素时，该段落需生成知识点。

具象知识元素（出现任一即生成）：
- A 层（疾病描述层）：全部
- B 层（文献现状层）：全部
- F 层（结果详述层）：带具体数据的元素
- D 层中的：新兴技术方法、现有技术现状、转化医学进展

套话元素（段落仅含这些时跳过）：
- D 层（方法概述层）除上述三个之外的全部元素
- C 层中的：研究目的、研究问题、研究假设、研究创新点、研究必要性、科学研究意义、临床研究意义、公共卫生意义、研究局限性、研究展望、概念理论框架、知识缺口、方法学缺陷
- E 层全部（结果概述层）
- G 层全部（研究结论层）
- 合规声明、方法贡献声明
