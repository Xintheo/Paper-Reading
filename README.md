# 摘要阅读自动化 — 模板仓库

本仓库存放 Obsidian 摘要阅读自动化系统的模板文件，由软件自动下载部署。

## 目录结构

```
.obsidian/          ← Obsidian 配置（snippets、快捷键等）
0-辅助文件/
  └── 2-Template/   ← Templater 模板脚本
1-AI指令/           ← AI 分析指令模板
2-辞典/             ← 摘要辞典 + 元素辞典
3-中转站/           ← AI 结果中转文件
```

## 使用方式

由配套桌面软件自动下载和更新，无需手动操作。

## 版本

查看 `version.json` 获取当前版本信息。
