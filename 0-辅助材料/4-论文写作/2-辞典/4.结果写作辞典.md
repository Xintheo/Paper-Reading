# 结果辞典 v3

## 标题名称

> 归一化识别 Results 二级标题的各种写法，用于定位 ## Results 起始行。格式：归一化名称: 变体1, 变体2, ... 匹配时自动剥离序号。

## 写作元素

> 切片按写作元素分类，作为未来写作时的检索 key。第二阶段 AI 输出时，每条切片必须标注其中一种元素。写作时按元素调用对应句式语料，模仿不同 PMID 的真实表达。

| 元素名 | 判断标准 | 典型例句 |
|--------|----------|----------|
| 总体描述 | 对整段实验或整组数据的概括性陈述，不涉及具体数字 | "Transcriptomic analysis revealed differential gene expression and pathway enrichment in the hypothalamus." |
| 方法说明 | 在结果章节里简要交代实验设计、分组、参数，作为该段实验的开篇。**含具体内容**（工具名、样本数、参数等） | "To elucidate the molecular mechanisms underlying PI, we first performed transcriptome sequencing on skin samples obtained from normal rats (Normal, n = 3) and PI rats (Injury, n = 3)." |
| 数据呈现 | 报告具体数字、阈值、基因/蛋白数量、样本数等**可量化结果**。重心是报告数据本身 | "Differential expression analysis identified 1,864 downregulated and 1,960 upregulated genes (\|log2fold change\| > 0.5 and p < 0.05)." |
| 统计说明 | 报告**统计学检验方法及其指标**（OR、CI、F统计量、AUC、Steiger等）。与数据呈现的区别：数据呈现报告实验结果数值，统计说明报告统计检验的方法和结论 | "Steiger directionality analysis confirmed the assumed causal direction (Steiger p < 0.001 for all proteins)." |
| 分别描述 | 列举多个独立发现，常见于功能富集、通路富集结果。用逗号/分号连接多项，无总概括 | "GO enrichment analysis showed that these DEGs were mainly involved in muscle tissue development, wound healing, and extracellular matrix organization." |
| 对比描述 | 明确对比两组及以上的差异（组间、剂量间、时间点间），含比较结构词 | "The CUMS group showed a marked reduction in cell viability compared with the Control group." |
| 图表引用 | 专门引导读者查看图表的句子，**去掉图表编号后句子不完整**。与数据呈现的区别：数据呈现的句子去掉图表编号后仍完整 | "As shown in Figures 1G,H, 476 genes exhibited reverse expression patterns." |
| 小结/转折 | 段落或章节结尾的概括，或转向下一发现的衔接。**含具体科学结论** | "These findings collectively confirm the successful induction of severe hepatocellular injury by chronic alcohol administration." |

### 写作元素边界裁决

> 当一条句子可能属于两个元素时，按以下规则裁决：

| 冲突场景 | 裁决 |
|----------|------|
| 数据呈现 vs 统计说明 | 报告实验产出的数值（基因数、浓度、倍数等）= **数据呈现**；报告统计检验的方法和结论（OR、CI、Steiger p值等）= **统计说明** |
| 数据呈现 vs 图表引用 | 去掉图表编号后句子仍完整 = **数据呈现**；不完整 = **图表引用** |
| 方法说明 vs 数据呈现 | 主要描述"用了什么做了什么" = **方法说明**；主要报告"得到了什么" = **数据呈现** |
| 分别描述 vs 小结/转折 | 列举占主体 = **分别描述**；总结/判断占主体 = **小结/转折** |
| 对比描述 vs 数据呈现 | 有明确比较结构（compared with, versus, 显著高于/低于）= **对比描述**；只是单组数据 = **数据呈现** |
| 数据呈现 vs 统计说明中的 p 值 | 句子主体是报告实验结果，p 值只是附带的显著性标注 = **数据呈现**；句子主体是报告统计检验过程和结论 = **统计说明** |

## 易混淆句式归类辅助

> **用途**：辅助语料收集和笔记整理时，将容易混淆的句式正确归入写作元素。所有句子都必须归入8种写作元素之一，不存在"忽略"或"跳过"的情况。

| 句式特征 | 归入写作元素 |
|----------|-------------|
| "To investigate X, we performed Y using Z" — 含具体工具和对象 | **方法说明** |
| "To further explore..." + 具体方法/工具/参数 | **方法说明** |
| "Next, we explored the effects of..." — 不含具体工具/参数 | **方法说明**（交代下一步实验） |
| "We further investigated..." — 不含具体内容 | **方法说明**（同上） |
| "The next part of our study focused on..." | **方法说明**（引出下一段实验） |
| "It has been reported that..." + 文献背景 | **总体描述**（引用已知事实作为背景铺垫） |
| "Consistent with previous reports..." + 本研究数据 | **数据呈现** 或 **小结/转折**（按主体内容判断） |
| "Given the significant prevalence of X, this finding underscores..." | **小结/转折**（含具体科学结论） |
| "Consistent with these processes, key terms included..." | **分别描述** 或 **数据呈现**（按内容判断） |
| "These findings collectively confirm..." | **小结/转折** |
| "As shown in Figure X, ..." — 去掉图表编号后句子不完整 | **图表引用** |
| "As shown in Figure X, ..." — 去掉图表编号后句子仍完整 | **数据呈现**（图表只是附带引用） |

## 研究套路映射

> **架构说明**：
> - 套路分为**基础套路**和**组合套路**两层
> - **基础套路**：原子单元，不可再拆，每个基础套路有独立的定义、涉及方法和常用图形
> - **组合套路**：由多个基础套路串联而成，名称用 ` + ` 连接，涉及方法和常用图形从基础套路**继承并集**，不单独定义
> - **颗粒度原则**：方法相同但输入数据/参数不同的，合并为一个基础套路；具体参数差异在写作时通过参数表体现
>
> 数据流向：
> - 第二阶段 AI 输出后，脚本自动写入新套路、合并已有套路的涉及方法和常用图形（并集合并）
> - AI 输出时，优先匹配此表中已有的套路名、方法名、图形名
>
> AI 输出新增标记规则：
> - 新套路 → 标 [新增套路]
> - 新图形 → 标 [新增图形]
> - 涉及方法必须严格匹配材料和方法辞典，匹配不上的标 [新增方法:xxx]
